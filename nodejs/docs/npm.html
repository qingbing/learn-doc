Node.js - npm 包管理工具

1. 什么npm 包管理工具 ？

npm ： node package manager, 基于nodejs的包管理工具。

2. 什么 包 ？

在nodejs中：包 === 项目；模块 === 文件

3. npm 常用命令
  1. npm install ： 自动查找当前目录下的package.json文件，安装里面所有的依赖
  2. npm install {包名称} ： 安装指定包
  3. npm i {包名称} ： 是npm install 的简写
  4. npm i {包名称}@{版本号} ： 安装指定包的指定版本
  5. npm i {包名称} -g ： 全局安装，window下路径为 用户下\AppData\Roaming\npm\node_modules
  6. npm i {包名称} --save ： 将安装包写入package.json的依赖列表
  7. npm i {包名称} --save-dev ： 将安装包写入package.json的开发时依赖列表
4. npm 其他命令
  1. npm search {包名称} ： 搜索包
  2. npm view {包名称} ： 查看包信息
  3. npm uninstall {包名称} ： 卸载包
  4. npm update {包名称} ： 更新包
  5. npm i {包名称} ： 是npm install 的简写
  6. npm i {包名称}@{版本号} ： 安装指定包的指定版本
  7. npm i {包名称} -g ： 全局安装，window下路径为 用户下\AppData\Roaming\npm\node_modules
  8. npm i {包名称} --save ： 将安装包写入package.json的依赖列表
  9. npm i {包名称} --save-dev ： 将安装包写入package.json的开发时依赖列表
5. cnpm 命令替代npm

npm 是一个文件下载工具，默认情况是去npmjs.com（github.com）下载资源

cnpm 是因为在国内下载npmjs.com的数据非常慢，所以淘宝制作了一个npmjs.com的镜像（可以下载，不能上传），是直接访问国内的缓存资源，cnpm会约10分钟同步一次npmjs的资源

安装 ： npm install -g cnpm --registry=https://registry.npm.taobao.org

使用 ： 和npm的命令一样
