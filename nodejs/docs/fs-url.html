Node.js - URL系统模块 url

1. 认识 url 模块
  1. 什么是URL ？
    全球统一资源定位符(Uniform Resource Locator)，对网站资源的一种简洁表达行驶，也成为网址
  2. URL的构成
    1. 完整结构
      协议://用户名:密码@主机名.名.域:端口号/目录名/文件名.扩展名?参数名1=参数值1&参数名1=参数值1#hash
    2. http协议的URL常见结构
      协议://主机名.名.域/目录名/文件名.扩展名?参数名1=参数值1&参数名1=参数值1#hash
      https://www.baidu.com:443/img/xxx
      http://www.baidu.com:80/img/xxx
