Node.js - 缓存区-buffer

1. 什么是缓存区

在内存中开辟的一个临时区域，用于存放需要运算的字节码

2. 如何创建缓存区

由于在测试过程中，发现new Buffer()，已经已经建议废弃，这里根据后期的制作，参考 buffer的实际使用

3. 写入缓冲区
// 向缓存区写入数据
buf.write('张三'); // 在nodejs中，默认使用utf-8编码，一个中文占三个字节
4. 读取缓冲区
// 读取缓存区
let out = buf.toString();
5. 复制缓冲区
// 创建指定长度的缓存区
let buf = Buffer.alloc(10); // 创建5个字节的缓存区
let buf1 = Buffer.alloc(20);

buf.write('张三');
// 复制缓冲区，是一个新的副本，将buf的内容复制给buf1
buf.copy(buf1);

// 打印buffer
console.log(buf);
console.log(buf1);
