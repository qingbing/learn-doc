Node.js - package.json 的使用

1. 什么是package.json

package.json 是node.js项目的包描述文件，以json格式的形式描述项目

2. 如何创建package.json
  1. npm init ： 每一步需要用户手动填写相关信息
  2. npm init -y ： 自动全部以yes的形式生成package.json
3. package.json 的常用属性
  1. name ： 项目名称
  2. version ： 项目版本，三段式
  3. homepage ： 包所在的官网，url
  4. author ： 包作者姓名
  5. keywords ： 关键字，发布到npm后，供其他人搜索的关键词
  6. repository ： 包代码的存放的类型，可以是git或svn
  7. contributors ： 其他贡献者
  8. description ： 项目描述
  9. main ： 主模块
  10. dependencies ： 依赖列表
  11. devDependencies ： 开始时依赖列表
  12. scripts ： 脚步命令，可以使用npm命令进行执行
  13. license ： 开源协议，很多使用MIT
