Node.js - nodejs 的作用域

1. nodejs 的作用域

由于nodejs在执行时会将所有的代码封装在函数中，所以，用户定义的变量或函数都是局部的，要将数据通过module.exports.xxx共享给其他模块使用

如果想使用全局的变量，使用global，eg：

// 定义全局变量
global.变量名 = 值或函数;
// 使用
let lname = global.name; // 或者
let lname = name; // 或者
