Node.js - path系统模块 path

1. 认识 path 模块
  1. 什么是 path 模块？
    path 模块是nodejs中提供的一个系统模块，用于格式化或拼接一个完整的路径
  2. path 模块的常用方法
    1. path.join(pa[, pa][, ...]) : 将多个字符串拼接成一个完整的路径
    2. path.dirname(pa) : 返回路径中代表文件夹的部分
    3. path.basename(pa) : 返回路径中的最后一部分，包含文件名和扩展名
    4. path.extname(pa) : 返回路径中的后缀部分，及最后一个"."后的部分，如果路径中不包含"."，或该路径只有一个"."，且这个点为路径的第一个字符，返回空字符串
    5. path.parse(pa) : 返回路径字符串的对象
    6. path.format(paObj) : 从对象中返回路径字符串，和path.parse 相反
  3. path 模块的不常用方法
    1. path.normalize(pa) : 格式化（简化、美化）路径
    2. path.isAbsolute(pa) : 判断pa是否为绝对路径
    3. path.resolve([from ...], to) : 将to参数解析为绝对路径
    4. path.relative(from, to) : 将路径转换为相对路径
  4. path 模块的几个属性
    1. path.sep : 平台文件路径的分隔符
    2. path.delimiter : 平台特定的路径定界符（路径于路径之间的分割）
2. path.join()
// 声明path系统模块
let path = require('path');

// 拼接路径
var pa = path.join(__dirname, 'xx');
console.log(pa); // /mnt/hgfs/Vhosts/learn.us/learn-doc/nodejs/demo/sub/xx
var pa = path.join('web', 'xx', 'yy');
console.log(pa); // web/xx/yy
3. 常用方法测试
// 声明path系统模块
let path = require('path');

var pa = './data/note.txt';
console.log(path.dirname(pa)); // ./data
console.log(path.basename(pa)); // note.txt
console.log(path.extname(pa)); // .txt
console.log(path.parse(pa));
/*
output :
{
  root: '',
  dir: './data',
  base: 'note.txt',
  ext: '.txt',
  name: 'note'
}
 */
