Node.js 安装和使用

1. 安装
# 下载
wget https://cdn.npm.taobao.org/dist/node/v12.13.1/node-v12.13.1-linux-x64.tar.xz

# 解压，进入目录
tar -xvf node-v12.13.1-linux-x64.tar.xz
cd node-v12.13.1-linux-x64

# 先备份，养成修改重要文件之前备份的习惯
cp /etc/profile /etc/profile.bak

# 修改环境变量
vim /etc/profile; 在末尾加入："export PATH=$PATH:{{nodejs-root}}/bin"

# 让环境变量立即生效
source /etc/profile

# 查看是否安装成功
node -v
2. 运行模式

  1. repl模式 ： 命令行（控制台）执行模式

    在命令行中输入"node"命令，在里面写入代码，类似"mysql的命令行模式"
# 退出
ctrl+C 两次

  2. 文件运行模式

    # 在命令行中直接执行js文件
node filename
