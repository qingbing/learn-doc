什么是nodejs？

  1. 一个基于 Chrome V8 引擎的 JavaScript 运行环境
  2. 与浏览器上的javascript执行功能一样
  3. 是一门编程语言

为何要学nodejs？

  1. 前后端语言统一
  2. 前后端分离
  3. 前端开发框架很多都是基于nodejs
  4. 上手快，开发简单
