Vue 实例化结构
                    new Vue({
                        el : '',
                        data : {}, // 显示 —— {{propertyName}}
                        computed : {}, // 计算属性，显示 —— {{methodName}}
                        filters : {}, // 过滤，使用 —— | methodName (methodName(arg...))
                        methods : {}, // 事件，执行 —— {{methodName()}}
                        watch : {}, // 属性监听
                        components : { // 局部组件
                            componentName : {
                                props : ['propertyName1', 'propertyName2', ...], // 父组件的数据需要通过 props 把数据传给子组件，子组件需要显式地用 props 选项声明 "prop"
                                template : '显示的html模版',
                            }
                        },
                    });
  1. 可以使用 methods 来替代 computed，效果上两个都是一样的，但是 computed 是基于它的依赖缓存，只有相关依赖发生改变时才会重新取值。而使用 methods ，在重新渲染的时候，函数总会重新调用执行
