Vue 指令

0. 什么是指令

是Vue中带有 v- 前缀的特殊属性，用于在表达式的值改变时，将某些行为应用到DOM上

1. v-html —— data数据绑定：用户定义的变量是html，用于输出html代码
    <p v-html="content"></p>
2. v-model —— data数据绑定：数据双向绑定，用于 input、textarea、select、checkbox、radio等表单控件
基本使用示例
3. v-if —— 判断指令：在html的document中被移除
<p v-if="seen">可以看见我</p>
4. v-show —— 是否显示指令：在html的document中被隐藏，其实就是 style="display:none"
<p v-if="seen">可以看见我</p>
5. v-bind —— 属性绑定指令
完整语法
<a v-bind:href="taobao.url" v-bind:target="taobao.target">{{taobao.label}}</a>
简写语法
<a :href="taobao.url" :target="taobao.target">{{taobao.label}}</a>
5. v-for —— 循环显示
<ol>
    <li>
        <!-- object or array -->
        <span v-for="val in info">{{val}},</span>
    </li>
    <li>
        <!-- object or array -->
        <span v-for="(val,index) in arr">{{index}}:{{val}},</span>
    </li>
    <li>
        <!-- object -->
        <span v-for="(val,key,index) in info">{{index}}:{{key}}-{{val}},</span>
    </li>
</ol>
5. v-on —— 事件绑定
<button v-on:click="clickButton('click button')">Click Button : click事件在vue.methods中</button>
<button @click="clickButton('click button')">Click Button : click事件在vue.methods中</button>
