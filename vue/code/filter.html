Vue 过滤器

0. 什么是指令

Vue允许用户自定义过滤器，用于一些常见的文本格式化，定义在filters中，使用时用 | 管道符来执行

1. 过滤器: | —— 使用示例
let myVue = new Vue({
    el: '#app',
    data,
    filters: {
        // 首字母大写，单词之间用"_"连接的过滤器
        formatId(val) {
            let tt = val.split(" ");
            let b = [];
            tt.forEach(function (value, index) {
                b.push(value.charAt(0).toUpperCase() + value.slice(1));
            });
            return b.join('_');
        }
    }
});

使用

{{testMessage | formatId}}
<div :id="id | formatId"></div>
