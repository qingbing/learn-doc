Vue 基础

1. 属性
  1. 定义：data.propertyName
  2. 基础显示：{{propertyName}}
  3. javascript 表达式显示：{{propertyName + 1}}、{{propertyName ? true : false}}...
  4. 修改属性值： 
    data.propertyName = '数据修改';
// 或者
myVue.propertyName = '实例修改';
2. 系统属性

除了数据属性，Vue实例提供了实例属性和方法，为了和用户自定的属性区分，它们都有前缀 "$"

  1. 元素 ： myVue.$el
  2. 数据 ： myVue.$data
3. 函数
  1. 定义：methods.functionName : (){}
  2. 执行：{{functionName()}}
