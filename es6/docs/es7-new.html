ES7——新增语法

1. async 函数
  1. 概念 : 真正意义上去结局异步回调的问题，同步流程表达异步操作
  2. 本质 : Generator的语法糖
  3. 语法
        async function foo() {
        await 异步操作;
        await 异步操作;
    }
  4. 特点
    1. 不需要想Generator那样去调用next方法，遇到await就等待，当前的异步操作完成就往下执行
    2. 返回的总是 Promise 对象，可以用then方法进行下一步操作
    3. async取代generator函数的星号 * ，await 取代Generator函数的yield
    4. 语意上更为明确，使用简单，就目前而言，尚无不良反应
    5. await 返回值
  5. ES7 —— async 真正的异步代码同步化
2. ** 指数运算符
console.log(3 ** 4); // 81
3. Array.prototype.includes(value) 判断数组中是否包含指定value
let arr = [2, 4, 5, 7, 4, 7];
console.log(arr.includes(4)); // true
console.log(arr.includes(8)); // false
